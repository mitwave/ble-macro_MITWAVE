# Nature Remo 非公式機器拡張マクロ集

## 概要

Nature Remo3/mini2/Lapisで使用可能な、サードパーティのBluetooth Low Energy対応デバイスを操作するための機器拡張マクロの定義です。

## 使い方

各マクロのディレクトリには、使用方法と設定手順を記載したREADMEが含まれています。マクロごとの特有の設定や要件を確認してください。

* [sb](./sb/) - SBのデバイス向けマクロ定義
